
1) Open Github.com 

2) Create Project/repository

3) Create branches

4) open git bash

5) go to project path using cd or right click in project folder to open git bash

6) git init : to initialize git

7) git status : to check for untracked files

8) nano .gitignore : creating git ignore files to ignore particular file or folder

9) add folder path with forward slash "/" for directory and save with ctrl+x and Y

10) go to github > click on code(green in color) > copy path

11) git remote add origin "https://github.com/niru2601/django.git" : add github url to local 
 ----------- git remote set-url origin "https://github.com/newbko2d/django.git" : chnage repository path 


12) git remote -v : to check url added or not 

13) git add . : '.' inidcates add file > git add used to add files form working to stagging area

14) git commit -m "comment" : commit the code and write the message in double quites ""

13) git checkout -b 'new_branch_name' : to create new branch in gitbash

14) git push origin main : push the code to github

15) git push origin main --force : rewrite git local history "Commits"

16) git checkout "branch_name" : switch between two branches

17) git branch -D "branch_name" : to delete branch form gitbash 

18) go to github and create new pull request > select branches from where you need to pull the code
